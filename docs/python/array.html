    # 列表是什么？

       列表是由一系列按特定顺序排列的元素组成。
    1

    # 列表初始化

       variable = [ 'value' ];
    1

    # 我们该如何访问列表中的元素？

       列表中的元素是存在一定顺序的，所以我们可以通过元素的所在位置的序号，对其进行访问（这里需要注意的是元素的序号，是从0开始计算的。也就是说要访问列表的第一个元素，那应该访问的是序号为0的列表元素）。
    1

    一般数据都有增删改查几种操作

    1. 增

      • 尾部追加 array.append(value);
      • 中间插入 array.insert(index, value);

    2. 删

      • 使用del删除 del array[index]
      • 使用pop删除 array.pop(index),如果括号内没有index，那默认会将列表的最后一个元素从列表中删除，并且pop函数会将删除的元素作为返回值返回
      • 在列表中删除特定的元素值，使用array.remove(value)方法。需要注意的是，array.remove(value) 只会删除第一个元素值(在同一个元素值出现多次的数组中)

    3. 改

      • 排序。排序有两种情况：array.sort()对列表永久排序 , sorted(array)临时排序（sorted函数返回排序好的一个列表，不会修改原本列表的顺序）
      • 逆序。逆序array.reverse(),会永久改变列表的顺序, 按照原本列表顺序的逆序进行排列。
      • 切片。需要列表中一部分,就需要用到切片。array[startIndex : endIndex] 会返回一个从原列表startIndex到endIndex的新列表（需要注意返回的列表不会包含endIndex的元素）.

    4. 查

      • 查询列表长度。len(array)的返回值就是列表的长度
      • 遍历列表。使用for ... in ...
      • 比较特殊的函数,例如min,max,sum,可以直接对数组中的最大值,最小值,总数进行统计.

    5. 创建

      • 使用range()来创建。range可以传3个参数,分别是起始值,末尾值(遍历出的值不会包含末尾值),间隔值(默认为1,可不传入).(python3中需要添加list(range()),list函数最后的返回值就是一个列表).range函数在python2和python3中返回值是不同的.当你使用python注意现在使用的版本类型,python2会返回一个列表,但python3会返回一个不变序列类型.不变序列类型还需要一个list(),才能将不变序列类型转换成列表.
        squares = [];
    for value in range(1,10):
        squares.append(value**2);

    1. 解析
      • 列表解析只是有一种更加特殊的创建列表的方式.创建的列表和上述相同.
        squares = [value**2 for value in range(1,10)];
    1

    元组是一个特殊的列表,它是不能修改的(但是它可以被重新定义).
同时 元组 = ()表示方法与列表也不太一样.
    1. 复制
      • 列表的复制是由切片来完成的.复制一个列表直接是使用 ar = array[:] 来完成的,这样得到的数组和原来相同,两者没有联系(深拷贝)
